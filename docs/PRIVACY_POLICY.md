# Privacy Policy - AI Query Hub

**Last Updated:** February 6, 2026

## Introduction

AI Query Hub ("we," "us," or "our") operates as a Living AI Assistant that provides autonomous, proactive AI services across multiple communication channels. This Privacy Policy explains how we collect, use, protect, and share information when you use our services.

By using AI Query Hub, you agree to the collection and use of information in accordance with this policy.

### 1. Account Information
**What we collect:**
- Email address (required for account creation)

**Documents and Files:**

**Goals and Preferences:**
- User-defined goals, tasks, and objectives
- Communication preferences and settings
- Proactive assistance preferences and schedules
- Autonomy session configurations

**Why we collect it:**
- Provide AI assistance and document intelligence services
- Maintain conversational context for better assistance
- Enable proactive and autonomous features
- Improve service quality and relevance

### 3. Integration Data
**Third-Party Connections:**
- OAuth tokens for Google Drive, Slack, and other connected services
- Telegram user ID and chat information (when connected)

**Why we collect it:**
- Enable multi-channel AI assistance
- Sync documents from connected services
- Provide seamless experience across platforms

### 4. Usage Analytics
**Technical Data:**
- IP address and general location (city/country level)
- Browser type, version, and device information
- Page views, feature usage, and interaction patterns
- Error logs and performance metrics
- API usage statistics and response times

**AI Interaction Data:**
- Query types, frequency, and complexity
- AI model usage (Primary/Fast/Cheap tier selection)
- Response quality feedback and ratings
- Feature adoption and usage patterns

**Why we collect it:**
- Monitor system performance and reliability
- Optimize AI model selection and costs
- Identify and fix technical issues
- Improve user experience and feature development

### 5. Proactive Assistance Data
**Behavioral Patterns:**
- Usage timing and frequency patterns
- Task and goal progress tracking
- Communication channel preferences
- Urgency indicators and response rates

**Autonomous Actions:**
- Proactive check-in triggers and outcomes
- Autonomous session logs and actions taken
- User confirmation and feedback on proactive assistance

**Why we collect it:**
- Enable intelligent proactive assistance
- Learn optimal timing for check-ins
- Improve autonomous decision-making
- Respect user boundaries and preferences

### 1. AI Model Processing
**Data Sent to AI Services:**
- Your queries and conversation context

- Conversation summaries for context maintenance

**AI Service Providers:**

### 2. Local Processing
**Data Processed Locally:**
- Document parsing and text extraction
- Search indexing and vector embeddings
- Usage analytics and performance metrics
- Backup and recovery operations

### 3. Real-Time Features
**Web Search Integration:**

## Data Security

### Technical Safeguards
**Encryption:**
- All data encrypted in transit using TLS 1.3
- Database encryption at rest using AES-256
- API communications secured with industry-standard protocols

**Access Controls:**
- Row-level security ensuring data isolation
- Multi-factor authentication for administrative access
- Regular security audits and penetration testing
- SOC2 Type II compliance through infrastructure partners

**Data Isolation:**
- Your data is isolated from other users at the database level
- AI processing uses session-based isolation
- Cross-user data access is technically impossible

### Operational Safeguards
**Monitoring:**
- 24/7 security monitoring and alerting
- Automated threat detection and response
- Regular security training for all team members
- Incident response procedures and regular drills

**Backup and Recovery:**
- Daily automated backups with encryption
- Point-in-time recovery capabilities
- Geographically distributed backup storage
- Regular disaster recovery testing

## Data Retention

### How Long We Keep Your Data
**Active Accounts:**
- **Conversations:** 30+ days for context, or until you delete them
- **Documents:** Until you delete them or close your account
- **Usage Analytics:** 24 months for service improvement
- **Audit Logs:** 7 years for security and compliance

**Deleted Accounts:**
- All personal data deleted within 30 days
- Some anonymized usage statistics may be retained
- Legal or security-related data may be retained longer as required

### Data Deletion
**You Can Delete:**
- Individual conversations and documents anytime
- Your entire account and all associated data
- Specific integration connections (Telegram, Slack, etc.)
- Upload history and processing results

**Automatic Deletion:**
- Temporary files (voice recordings, processing cache) within 24 hours
- Expired authentication tokens immediately
- Inactive account data after 2 years of inactivity (with prior notice)
